# MotionCore App Icon Oluşturma

## Adımlar:

1. **Icon'u oluştur:**
   - Tarayıcıda `assets/icon/create_icon.html` dosyasını açın
   - "Download Icon (1024x1024)" butonuna tıklayın
   - İndirilen dosyayı `assets/icon/app_icon.png` olarak kaydedin

2. **Icon'ları uygula:**
   ```bash
   flutter pub run flutter_launcher_icons
   ```

3. **Uygulamayı yeniden derle:**
   ```bash
   flutter clean
   flutter run
   ```

## Icon Özellikleri:
- Grey Rock gezegeni (soldaki ekrandaki gibi)
- Krater detayları
- "MotionCore" yazısı (cyan renk)
- "TERRAFORM" alt yazısı
- Uzay temalı arka plan (yıldızlı)
