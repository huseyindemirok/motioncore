# MotionCore App Icon

Uzay temalı app icon oluşturmak için:

1. Bu klasöre `icon.png` dosyası ekleyin (1024x1024 px)
2. Veya `flutter pub run flutter_launcher_icons` komutunu çalıştırın

Icon özellikleri:
- Uzay temalı (karanlık arka plan, yıldızlar)
- Merkezde neon mavi gezegen
- "MC" harfleri veya MotionCore logosu
- Cyan/Blue renk paleti
